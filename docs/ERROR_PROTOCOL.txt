Ошибка - наличие строк в таблице ошибок (mistakes) при нажатии кнопки "следующий шаг"

Завершение шага - достижения нужного позиции элемента (conditon_order в таблице element_condition_positions)

2 вида правильного шага:
    проверяем condition_order, 
    - если не совпадает с картой, то недокрутил => ждем пока докрутит и ничего не делаем
    - если совпадает, то добаляем поля в таблицу выполненных шагов (finished_steps): step_id - id выполненного шага, exercise_status_id - id указывающий на текущее состояние прохождения карты 

2 вида ошибок: 
 - перекрутил - удаляем запись о этом шаге из таблицы выполненных шагов (finished_steps)


 - нажал не туда (добавляем в таблицу ошибок (mistakes) следующие поля: condition_group_id (нужный тип элемента), 
                                                                      condition_order (номер позиции в этом типе элемента),
                  при повторном нажатии на этот элемент, заходим в таблицу ошибок (mistakes):
                  - если condition_order совпадает с тем, что в талице - ошибка исправлена и удаляем строчку из таблицы
                  - если condition_order не совпадает, то ничего не делаем, а ждем пока пользователь не исправится
                  )
